Method 2: VS Code Live Server

üîç Understanding Your Code

HTML Structure (index.html):

  • <nav> - Navigation menu at top

  • <footer> - Bottom of page

  • Colors are defined with codes like #2d8a3e
  • padding adds space inside elements
  • margin adds space outside elements
  • @media makes it work on mobile

  • Smooth scrolling when clicking menu links
  • Button click handlers
  • Console messages (Press F12 to see them!)
